## 📌 프로젝트 소개

**Playstyle Profiler**는 게임 플레이 방식을 분석하여 MBTI 성격 유형과 연결하는 인터랙티브 웹 애플리케이션입니다.

16개의 핵심 질문을 통해 사용자의 게임 플레이 스타일을 분석하고, 이를 바탕으로 MBTI 기반의 맞춤형 게임 추천, 전략, 무기 등을 제공합니다.

### 🎯 핵심 목표

- 게이머가 자신의 플레이스타일을 객관적으로 이해할 수 있도록 지원
- MBTI 성격 유형과 게임 플레이 방식의 상관관계 분석
- 개인화된 게임 추천 및 전략 제공으로 게임 경험 향상

---

## ✨ 주요 기능

### 1. 📝 인터랙티브 퀴즈 시스템

- 16개의 핵심 질문으로 구성된 체계적인 분석
- 6개 카테고리: 게임플레이 스타일, 팀플레이, 문제 해결, 게임 선호도, 사회적 상호작용, 성취 방식
- 실시간 진행률 추적 및 답변 수정 기능
- LocalStorage를 통한 진행 상황 자동 저장

### 2. 🧠 MBTI 분석 엔진

- 각 선택지에 대한 정밀한 점수 시스템
- 4개 차원(E/I, S/N, T/F, J/P) 기반 계산
- 16가지 MBTI 유형별 상세 프로필 제공

### 3. 🎮 맞춤형 추천 시스템

- **게임 추천**: MBTI 유형에 맞는 게임 장르 및 타이틀
- **무기/장비 추천**: 플레이스타일에 어울리는 게임 내 무기
- **전략 제공**: 유형별 최적 플레이 전략 및 팁
- **호환 유형**: 함께 플레이하기 좋은 다른 MBTI 유형

### 4. 📊 통계 대시보드

- 전체 사용자 통계 데이터 시각화
- MBTI 유형별 분포 차트
- Vercel KV 기반 실시간 데이터 업데이트

### 5. 🌓 사용자 경험

- 다크 모드 / 라이트 모드 지원
- 모바일, 태블릿, 데스크톱 반응형 디자인
- Framer Motion 기반 부드러운 애니메이션
- 결과 공유 기능 (소셜 미디어, 링크 복사)

## 🛠️ 기술 스택

### Frontend

| 기술 | 설명 | 버전 |
|------|------|------|
| **Next.js** | React 프레임워크 (App Router) | 15.x |
| **React** | UI 라이브러리 | 19.x |
| **TypeScript** | 정적 타입 언어 | 5.x |
| **Tailwind CSS** | 유틸리티 CSS 프레임워크 | 3.x |
| **Shadcn UI** | 컴포넌트 라이브러리 | Latest |
| **Framer Motion** | 애니메이션 라이브러리 | 12.x |

### State Management

- **Zustand**: 경량 상태 관리
  - DevTools 지원
  - LocalStorage Persist

### Backend & Storage

- **Vercel KV**: 서버리스 Redis 스토리지 (통계 데이터)
- **LocalStorage**: 클라이언트 사이드 저장 (진행 상황)

### Development Tools

- **ESLint**: 코드 품질 관리
- **Prettier**: 코드 포맷팅
- **React Hook Form**: 폼 관리
- **Zod**: 스키마 검증
